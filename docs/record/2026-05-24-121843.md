## 可能存在的隐患

- 完整根目录测试命令 `uv run pytest` 未通过：测试收集到 `services/hf_realtime_gateway/tests/test_transcription_dedupe_patch.py` 时，当前根环境缺少 `speech_to_speech`，报错 `ModuleNotFoundError: No module named 'speech_to_speech'`。该依赖属于 gateway 子项目 `services/hf_realtime_gateway/pyproject.toml`，不是根应用环境依赖。
- 本次替代验证为 `uv run pytest tests`，根应用测试通过 `255 passed in 24.26s`；同时执行 `git diff --check -- README.md README_zh.md`，无空白错误。
- OpenClaw bridge 当前是 text-only，`include_camera_image` 只会说明相机帧是否可用，不会发送图片字节；视觉分析仍应优先使用本地 `camera` 工具。
- Ark sidecar 工具路由依赖外部 OpenAI-compatible chat completions 服务；如果 `GATEWAY_LLM_BASE_URL` 或 `GATEWAY_LLM_MODEL` 缺失，Ark 仍可语音对话，但不会触发本地工具。
- `send_email` 涉及真实 SMTP 发送，运行环境必须使用应用专用密码或等价安全凭据，不能把真实密钥提交到仓库。
